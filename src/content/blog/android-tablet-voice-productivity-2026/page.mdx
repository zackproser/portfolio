## Setup Pro Tips

**Microphone Position**: External Bluetooth microphones (like AirPods Pro or Sony WF-1000XM4) dramatically improve accuracy in noisy environments. Position tablets where built-in mics aren't blocked by hands or surfaces.

**App Priority**: Set WisprFlow to high performance mode in Android settings. This prevents system resource conflicts during intensive voice processing.

**Backup Sync**: Enable automatic sync for all voice-generated content. Tablets are mobile devices—ensure work isn't lost if battery dies or device gets damaged.
