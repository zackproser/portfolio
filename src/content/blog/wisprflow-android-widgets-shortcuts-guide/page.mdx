## Floating Bubble Mode - Android's Secret Weapon

**What is Floating Bubble Mode?**
A persistent, moveable recording interface that hovers over any app:
- Stays accessible while using other apps
- Minimal screen real estate usage
- Tap to start/stop recording
- Drag to reposition anywhere on screen

**Setting Up Floating Bubble:**
1. Enable "Display over other apps" permission
2. Activate bubble mode in WisprFlow settings
3. Choose bubble size and transparency
4. Configure tap and long-press actions

**Bubble Customization:**
- Size: Small (24dp) to Large (72dp)
- Transparency: 20% to 100%
- Position memory (remembers where you place it)
- Auto-hide after inactivity
- Different bubble styles for different recording modes

**Advanced Bubble Features:**
- **Smart positioning:** Avoids keyboard and UI elements
- **Edge snapping:** Automatically aligns to screen edges
- **App-specific behavior:** Different bubble settings per app
- **Gesture controls:** Swipe gestures for quick actions

## Troubleshooting Android-Specific Issues

**Widget Not Responding:**
1. Check if WisprFlow has "Display over other apps" permission
2. Exclude WisprFlow from battery optimization
3. Clear WisprFlow cache and restart
4. Remove and re-add widget to home screen

**Quick Settings Tile Missing:**
1. Verify WisprFlow is updated to latest version
2. Check Android version compatibility (Android 7.0+)
3. Reset Quick Settings panel to defaults
4. Reinstall WisprFlow if issue persists

**Floating Bubble Performance:**
1. Reduce bubble size if experiencing lag
2. Increase transparency for better performance
3. Limit bubble to specific apps only
4. Check for memory usage in Android settings

**Automation Not Triggering:**
1. Verify Tasker has proper permissions
2. Check WisprFlow plugin installation
3. Test individual actions separately
4. Review trigger conditions and timing
