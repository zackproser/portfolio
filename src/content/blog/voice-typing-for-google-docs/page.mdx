## Google Docs Built-in Voice Typing

Google Docs includes a native voice typing feature that's free and easy to access. Here's what you need to know:

### How to Access It

1. Open any Google Docs document
2. Go to Tools > Voice typing (or press Ctrl+Shift+S)
3. Click the microphone icon to start dictating
4. Speak clearly and naturally

### Capabilities and Limitations

**What it does well:**
- **Free to use** with any Google account
- **Basic punctuation recognition** for periods, commas, and question marks
- **Multiple language support** for international users
- **Simple voice commands** like "new line" and "new paragraph"

**Where it falls short:**
- **Limited accuracy** with technical terms, names, and industry jargon
- **No custom vocabulary** training for specialized terminology
- **Basic formatting options** compared to dedicated voice typing tools
- **Requires Chrome browser** for optimal performance
- **No offline functionality** - requires internet connection
