### Android: RecyclerView Adapter

**Voice Input:**
"Create a RecyclerView adapter class named ProductAdapter. Take a list of Product objects in the constructor. Override getItemCount to return products size. Create a ViewHolder inner class with TextView fields for name and price."

**Generated Kotlin Code:**
```kotlin

    override fun getItemCount(): Int = products.size
    
    class ViewHolder(itemView: View) : RecyclerView.ViewHolder(itemView) {
        val nameTextView: TextView = itemView.findViewById(R.id.productName)
        val priceTextView: TextView = itemView.findViewById(R.id.productPrice)
    }
}
```
