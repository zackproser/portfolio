This course is for beginners. You can take it even if you have zero experience working with computers. 

This course will walk you through the basics of HTML and CSS and help you create and deploy a simple portfolio website
which will form the basis of your new professional online web presence. 

Future courses on this site will be designed to help you build ambitious projects to showcase on your portfolio website, 
in order to attract new opportunities and to use as an asset during your job search.

## Course outline

This course is designed to: 
1. Be accessible to beginners
2. Get you a working public website you can start building on and improving
3. Teach you the basics you need to iterate on your portfolio website

**Introduction to Web Development and Static Sites**
  * Explanation of static vs. dynamic websites.
  * Introduction to web development languages, focusing on HTML and CSS.

**Setting Up a Development Environment**
  * Tools needed (text editor, web browser).
  * Overview of Git and GitHub, link to GitHub docs for account creation.

**Building a Simple Portfolio Website**
  *  Creating the structure of the site with HTML.
  *  Styling the site using CSS.

**Version Control with Git**
  * Basic concepts of version control.
  * Steps to create a repository on GitHub.
  * How to commit and push changes.

**Deploying with GitHub Pages**
  * How to configure and publish a site using GitHub Pages.
  * Managing site settings and understanding how to update the site content.
