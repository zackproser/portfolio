## CSS Fundamentals

CSS (Cascading Style Sheets) is used to control the presentation of HTML elements. Here's a simple example to change the color and alignment of your heading and paragraph:

```css
h1 {
    color: blue;
    text-align: center;
}
p {
    color: gray;
    text-align: center;
}
```
